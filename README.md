# student-wellbeing-analysis

## Description

AI-Based Sentiment Analysis Through Writing Papers for Students Well-being will analyze and predict emotional issues. This project helps parents and educators to provide anticipated assistance to children with potential emotional issues.

### Technologies

JavaScript, NodeJs, Typescript, Nodemailer
Huggingface/inference and Vader for sentiment analysis
lucide-react for charts

### Framework

Next.js V15
TailwindCSS V4

### Database

MongoDB
